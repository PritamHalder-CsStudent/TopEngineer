# TopEngineer
TopEngineer is a frontend web application designed for a technical services company. 
The platform provides a professional online presence where clients can explore and request hardware, security, 
and digital solutions.The live website is deployed and accessible at the TopEngineer Live Site.
# Live Link 
https://top-engineer-tec.vercel.app
# Services offered
The platform showcases a variety of core technical services:
>Laptop and desktop repair and maintenance.
>CCTV installation and security configuration.
>Website development and design.
>Network installation, optimization, and ongoing maintenance.

# Technologies used 
The project is built entirely on a modern frontend stack:
1. React: A component-based JavaScript library used to build a dynamic and interactive user interface.
2. Tailwind CSS: A utility-first CSS framework utilized for modern, fully responsive, and rapid UI styling.
3. Cloudinary: Cloud-based image and video management service used to optimize and deliver media assets efficiently.
4. Vercel: Cloud platform optimized for hosting frontend applications, used for continuous deployment.
